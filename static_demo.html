Netlify Deployment Notice

This is a static demonstration of the FLEXXI Bot Control Panel interface.

The full application requires backend services that cannot run on Netlify.

For the complete functionality, please run the application locally or on a traditional server.

Bot Control

Manage emote bots and send commands to player accounts.

User Management

Admin panel for creating and managing user accounts.

Real-time Monitoring

Monitor bot status and player interactions in real-time.

Login Required

To access the full control panel, you would normally need to log in with valid credentials.

This static demo shows the interface but cannot process login requests.

Deployment Information

This application requires two continuously running services:

  • main.py - The main bot service
  • web_panel.py - The Flask web interface

Netlify does not support continuously running backend services.

For full functionality, deploy to a traditional server or cloud platform that supports Python applications.
